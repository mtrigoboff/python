pcc.gui Package API¶

pcc.gui.window.newWindow(name, keypressFn=None, updateFn=None)¶

Returns a new window object with specified name, keypressFn, and updateFn. The second and third arguments are optional. See Window Functions.

Parameters

  • name (string) – window name

  • keypressFn (Python function) – keypress handler (see Window Functions)

  • updateFn (Python function) – updates the user interface state (see Window Functions)

Returns

pcc.gui.Window instance

class pcc.gui.window.Window(title, keyPressFn=None, updateFn=None)¶

A Python class that creates a window on the screen which can be populated with graphical user interface items like text fields and buttons.

setPosn(x, y)¶

Sets the position of the window on the screen. The x and y arguments specify the position of the top-left corner of the window. The top-left corner of the screen is x == 0, y == 0. Larger values of x are to the right. Larger values of y are lower.

Parameters

  • x (int) – x coordinate of top-left corner of window

  • y (int) – y coordinate of top-left corner of window

Returns

nothing

setSize(width, height)¶

Sets the size of the window on the screen.

Parameters

  • width (int) – width of the window

  • height (int) – heightof the window

Returns

nothing

setResizable(widthResizable, heightResizable)¶

Makes the width and/or the height of the window resizable.

Parameters

  • widthResizable (Boolean) – make the width of the window resizable

  • heightResizable (Boolean) – make the height of the window resizable

Returns

nothing

addButton(name, btnClickFn, sameRow=False, enabled=True)¶

Adds a button to the window. Specify sameRow=True to add the button to the same row as the previously added window item.

Parameters

  • name (string) – name of the button

  • btnClickFn (fn(string)) – function to call when the button is clicked. The function argument will be the name of the button.

  • sameRow (Boolean) – whether to add the button to the current row or the next row

  • enabled (Boolean) – whether the button is enabled or disabled (grayed out)

Returns

nothing

addTextField(name, enabled=True)¶

Adds a text field to the next row of the window.

Parameters

  • name (string) – name of the text field

  • enabled (Boolean) – whether the text field will accept user input

Returns

nothing

getValue(name)¶

Gets the value of the named window item. Can cause a NotImplentedError for item types that do not contain a value (e.g. button). Can cause a NoSuchItemError if name is not the name of an item.

Parameters

name (string) – name of the window item

Returns

value of the item

Return type

depends on item type; string for text field

setValue(name, value)¶

Sets the value of the named window item. Can cause a NotImplentedError for item types that do not contain a value (e.g. button). Can cause a NoSuchItemError if name is not the name of an item.

Parameters

  • name (string) – name of the window item

  • value (depends on item type) – value for the window item

Returns

nothing

setEnabled(name, enabled)¶

Sets the enabled state of the named window item. For instance, a button could be grayed out and unresponsive or normal-looking and active. Can cause a NotImplentedError for item types that do not support enabled/disabled states. Can cause a NoSuchItemError if name is not the name of an item.

Parameters

  • name (string) – name of the window item

  • enabled (Boolean) – value for the window item

Returns

nothing

open()¶

Opens the window.

Returns

nothing

close()¶

Closes the window.

Returns

nothing

focusReturnKey()¶

Call this from your myKeypressFn(key) function if you want the window item that currently has the focus to respond to the Return/Enter key. See User Interface Item Focus and the Return/Enter Key and Window Functions.

Returns

nothing
